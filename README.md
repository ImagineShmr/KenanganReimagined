## ✨ Features

- **Modern UI/UX** - Clean, premium design with smooth animations
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **6 Main Pages** - Home, Menu, Locations, Our Story, Careers, Contact, Download App
- **Interactive Components** - Animated hero sections, category filters, location finder
- **Framer Motion Animations** - Smooth scroll reveals and transitions

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS with custom brand colors
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/KenanganReimagined.git

# Navigate to project directory
cd KenanganReimagined

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be running at `http://localhost:{PORT}`

### Build for Production

```bash
npm run build
```
